== Description ==

This plugin is useful for optimzing H1 headings on pages.

== Installation ==

1. upload this plugin via the WordPress admin as a zip or FTP in and upload to wp-content/plugins.

2. Activate.

3. Check and see if I broke the page edit screen :)
